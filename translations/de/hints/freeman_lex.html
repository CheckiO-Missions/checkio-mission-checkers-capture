Fangen Sie eine Idee 💡! Die maximale Anzahl von Figuren, die in einem einzigen Zug geschlagen werden können, lässt sich am besten mit einer Methode berechnen, die lokal durch die vier diagonalen Richtungen von der aktuellen Position des Königs aus Schleifen zieht. Für jede dieser Richtungen, die eine gegnerische Figur mit einem freien Feld dahinter enthält, entfernen Sie diese gegnerische Figur vom Brett und berechnen rekursiv die Anzahl der Figuren, die von dem freien Feld, auf das Ihr König springt, geschlagen werden können. Sobald dieser rekursive Aufruf erfolgt ist, setzen Sie die gefangene Figur wieder auf das Brett und fahren mit der Schleife zur nächsten diagonalen Richtung fort. Als Antwort wird die Anzahl der Schläge zurückgegeben, die in der jeweiligen Richtung von diesem Feld aus das beste Ergebnis liefern.
